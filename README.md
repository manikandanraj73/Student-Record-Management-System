# Student Record Management System
    A simple full stack application to perform CRUD operation on student records with role based access

# Tech Stack
    - Java
    - Spring Boot
    - MySQL
    - HTML, CSS, JavaScript

## Features
    - Login System
    - Role based access (Admin / Teacher / Student)
    - ADMIN can (Add,Delete,View,Update) 
    - TEACHER can (Add,View,Update) 
    - STUDENT can (View student details) 
    - REST APIs
    - Frontend Intergration

## How to Run
 ### Backend
 1. Open backend folder in Intellij
 2. Update DB config in application.properties
 3. Run Spring Boot application

  ### Fronend
 1. Open frontend folder
 2. Open index.html in browser

## API Endpoints
 - POST = /login
 - GET = /students
 - POST = /students
 - PUT = /students/update
 - DELETE = /students/delete
